# Secret Life of Blood Glucose

An interactive data visualization exploring blood glucose patterns across non-diabetic, pre-diabetic, and diabetic groups using the CGMacros dataset from PhysioNet.

## Quick Start

### Prerequisites
- [Node.js](https://nodejs.org/) (v16 or later)
- Python 3 (for data preprocessing)

### Data Setup
1. Download the [CGMacros dataset](https://physionet.org/content/cgmacros/1.0.0/) from PhysioNet
2. Unzip and rename the folder to `data`
3. Run `python data_cleaning.py` to process the raw data

### Run Locally
```bash
npm install
npm run dev
```
